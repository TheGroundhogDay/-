

# 欢迎使用 Golang并发爬虫系统

**本系统主要实现一种轻量级的「常用爬虫工具封装」**

#### 已实现的简易功能 list

- [x] 根据url下载图片
- [x] 并发（带信号量）下载图片
- [x] 从alt抓取图片名称
- [x] 从链接中抓取文件名
- [x] 使用时间戳+随机数命名文件
- [x] 使用alt+链接中的名字做文件名
- [x] 并发爬取分页组图
- [x] 并发爬取分页组图（使用管道）




> 点击[目录](https://github.com/TheGroundhogDay/spiderkitUtil/blob/master/list.txt)即可查看
